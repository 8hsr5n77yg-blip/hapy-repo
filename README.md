# PDF 阅读项目

## 使用方式

1. 将 PDF 文件放入 `pdfs/` 目录
2. 告诉 Claude 文件名和你想了解的内容
3. Claude 会直接读取 PDF（包括图片、图表、扫描件）

## 读取能力

- 文字内容：直接提取
- 图片中的文字/图表：视觉识别（多模态）
- 扫描件 PDF：能识别图片中的文字
- 每次最多读 20 页，大文档分多次读取

## 大 PDF 处理

对于较大的 PDF（50 页以上），Claude 会分页段读取：
- `Read pdfs/doc.pdf pages: "1-20"` — 读前 20 页
- `Read pdfs/doc.pdf pages: "21-40"` — 读 21-40 页
- 以此类推
